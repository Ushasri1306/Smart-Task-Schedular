# Smart Task Scheduler

Smart Task Scheduler is a simple web application that helps users **create, manage, and prioritize tasks efficiently**. The application provides an easy-to-use interface where users can add tasks, set priorities, schedule deadlines, and manage their task list.

---

## 🚀 Features

* 📝 Add tasks with a title
* ⚡ Assign task priority (e.g., High)
* 📅 Select due date and time using a date-time picker
* ➕ Add tasks to a task list instantly
* ↩️ Undo recently deleted tasks
* 📋 View all tasks in a dedicated task section
* 🎨 Clean and modern user interface

---

## 🖼️ Application Interface

The application includes:

### Task Input Section

Users can enter task details:

* **Task Title**
* **Priority Selection**
* **Date & Time Picker**

Buttons:

* **Add Task** – Adds a new task to the list
* **Undo Delete** – Restores the last deleted task

### Tasks Section

Displays all created tasks.
If no tasks exist, the message **"No tasks yet."** is displayed.
